### Flutter gallery

官方MD风格demo，提供浏览各种控件的使用效果

**构建并安装app**

* `cd gallery`
* `flutter upgrade`(可选操作)
* `git clone https://flutter.googlesource.com/gallery-assets`
* `flutter run`

`flutter run --release`命令会编译并安装app（确保已连接设备）。
